Detailed Description

Function object for fitness as a function of individual mutations in a diploid.

Note
The updating policies must take a non-const reference to a double as the first argument and an mcont_t::value_type as the second. Further, they must not return anything. Any remaining arguments needed should be passed via a mechanism such as std::bind and a function object, or via a lambda expression. See KTfwd::multiplicative_diploid for an example implementation.
Examples:
migsel_ind.cc.
