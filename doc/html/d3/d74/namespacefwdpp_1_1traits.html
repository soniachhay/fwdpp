using fwdpp::traits::fitness_fxn = typedef traits::internal::fitness_fxn<dipvector_t, gcont_t, mcont_t>

Defines a struct with a single member typedef called type. If type is void, then one or more of dipvector_t, gcont_t, and/or mcont_t are not valid input types for a fitness function. Otherwise, type will evaluate to std::function<void(const dipvector_t::value_type,const gcont_t,const mcont_t)>

◆ mutation_model

template<typename mcont_t >
using fwdpp::traits::mutation_model = typedef typename traits::internal::mutation_model<mcont_t>::type

Gives the mutation model function signature corresponding to mcont_t. Applies to mutation policies that only take recycling bins and mcont_t as arguments.

If mcont_t is not a container of mutations, then mmodel_t will evaulate to void.

Otherwise, it will evaluate to std::function<std::size_t(recycling_bin_t<mcont_t> &,mcont_t &)>;

using fwdpp::traits::mutation_model_diploid = typedef typename traits::internal::mutation_model_diploid< diploid_t, mcont_t, gcont_t>::type

Gives mutation model function signature for models requiring diploids as arguments. If mcont_t is not a container of mutations and/or gcont_t is not a container of haploid_genomes, them mmodel_haploid_genome_t will evaluate to void.

Otherwise, it will evaluate to std::function<std::size_t(recycling_bin_t<mcont_t> &, const diploid_t &,const typename gcont_t::value_type &, mcont_t &)>;
