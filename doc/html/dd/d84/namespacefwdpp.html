◆ compact_mutations()

template<typename T >
std::vector<fwdpp::uint_t> fwdpp::compact_mutations ( T &  pop)

Sort mutation positions in population.

Reorders the population mutation container so that it is sorted by increasing mutation position. The reordering requires assigning new key values into all gametes, which is also done. The mutation counts and mut_lookup data structures also get updated.

Running this periodically is a performance increase due to improved cache performance.

Parameters
popA population.
Returns
A vector containing the mapping of old to new mutation index
Examples:
diploid_fixed_sh_ind.cc, and diploid_ind.cc.
