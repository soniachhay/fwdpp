Detailed Description

template<typename T, typename parameter_name>
struct fwdpp::strong_types::named_type< T, parameter_name >

Implementation of "strong types".

Allows the construction of strong/unambiguous interfaces via thin aliases.

At optimization levels like -O2 and higher, access via get will be completely optimized out.

Example use includes fwdpp::trait and fwdpp::fitness as arguments to fwdpp::additive_diploid and fwdpp::multiplicative_diploid.

Version
0.7.4 Added to fwdpp
Examples:
diploid_fixed_sh_ind.cc, diploid_ind.cc, juvenile_migration.cc, K_linked_regions_extensions.cc, K_linked_regions_generalized_rec.cc, spatialts.cc, and wfts.cc.
