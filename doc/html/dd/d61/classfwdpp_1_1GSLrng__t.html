Detailed Description

template<typename T>
class fwdpp::GSLrng_t< T >

A wrapper around gsl_rng * objects.

The template instantiation type must be a model of fwdpp::sugar::GSL_RNG_TYPE_TAG, which specifies the gsl_rng type. This type holds an object of type fwdpp::sugar::gsl_rng_ptr_t, which is a smart pointer that manages freeing the gsl_rng * upon destruction.

Examples:
diploid_fixed_sh_ind.cc, diploid_ind.cc, juvenile_migration.cc, K_linked_regions_extensions.cc, K_linked_regions_generalized_rec.cc, spatialts.cc, and wfts.cc.
