Detailed Description

template<typename T>
class KTfwd::GSLrng_t< T >

A wrapper around gsl_rng * objects.

The template instantiation type must be a model of KTfwd::sugar::GSL_RNG_TYPE_TAG, which specifies the gsl_rng type. This type holds an object of type KTfwd::sugar::gsl_rng_ptr_t, which is a smart pointer that manages freeing the gsl_rng * upon destruction.

Examples:
bneck_selection_ind.cc, diploid_binaryIO_ind.cc, diploid_ind.cc, diploid_ind_2locus.cc, and migsel_ind.cc.
