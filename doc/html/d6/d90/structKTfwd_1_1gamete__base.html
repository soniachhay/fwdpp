Detailed Description

template<typename TAG = tags::standard_gamete>
struct KTfwd::gamete_base< TAG >

Base class for gametes.

A gamete contains one container of keys to neutral mutations, and another to selected mutations. It also keeps track of its count (number of occurrences).

The template parameter types are: tag_type = A type that can be used as a "dispatch tag". Currently, these are not used elsewhere in the library, but they may be in the future, or this may disappear in future library releases.

Note
The typical use of this class is simply to define your mutation type (see Tutorial 1: Policies in fwdpp) and then use a typedef to define your gamete type in the simulations:
using gamete_t = KTfwd::gamete_base<mutation_type>
See Tutorial 1: Policies in fwdpp for examples of this.
Examples:
diploid_binaryIO_ind.cc, and migsel_ind.cc.
