Detailed Description

template<typename mutation_t>
struct KTfwd::mutation_reader< mutation_t >

Facilitates serialization of mutation types supported by the fwdpp sugar library.

The template parameter must be derived from KTfwd::mutation_base

Examples:
diploid_binaryIO_ind.cc, and migsel_ind.cc.
