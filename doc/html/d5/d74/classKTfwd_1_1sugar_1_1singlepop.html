Detailed Description

template<typename mutation_type, typename mcont, typename gcont, typename dipvector, typename mvector, typename ftvector, typename lookup_table_type>
class KTfwd::sugar::singlepop< mutation_type, mcont, gcont, dipvector, mvector, ftvector, lookup_table_type >

Abstraction of what is needed to simulate a single population using an individual-based sampler from fwdpp.

All that is missing is the mutation_type and the container types.

See Tutorial 5: the "sugar" layer on top of fwdpp for rationale, etc.

Examples:
migsel_ind.cc.
