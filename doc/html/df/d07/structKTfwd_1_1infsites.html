Detailed Description

Generic function object implementing the infinitely-many sites mutation model for "standard" population-genetic scenarios.

Examples:
bneck_selection_ind.cc, diploid_binaryIO_ind.cc, diploid_ind.cc, diploid_ind_2locus.cc, and migsel_ind.cc.
